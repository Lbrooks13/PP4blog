##  About the App
How Big Are Your Poos? is a quirky and engaging blog platform where users can share their unique bathroom experiences, comment on others' stories, and express appreciation through likes. Designed for fun and connection, How Big Are Your Poos? aims to create a lighthearted community around a universally relatable topic.

The app is intended for individuals who enjoy humor, storytelling, and connecting over shared experiences. It provides an interactive and user-friendly platform to share stories, read others' posts, and engage through comments and likes.

##  Features

### Existing Features

#### Navigation Bar
- Featured on all pages, the responsive navigation bar includes links to the Home page, Create Post page, and Login/Signup page.
- Provides consistent and easy navigation across the app on all devices.

#### Landing Page
- Welcomes users with a clean and humorous introduction to the app.
- Includes a call-to-action button for users to sign up or browse posts.

#### Post Feed
- Displays a list of stories uploaded by users in chronological order.
- Each post shows the title, an excerpt, the author, and the number of likes and comments.

#### Story Detail Page
- Provides the full story content along with a comment section for interaction.
- Includes a "like" button for users to show their appreciation.

#### User Authentication
- Allows users to register, log in, and log out securely.
- Provides personalized experiences, including the ability to create, edit, and delete their own posts.

#### Footer
- Contains links to social media pages and contact information.
- Provides a quick way for users to connect and stay updated.

### Features Left to Implement

1. **Search and Filter**: Allow users to search for posts by keywords or filter by popularity.
2. **User Profiles**: Add a user profile page showing their posts, liked stories, and activity.
3. **Tagging System**: Enable categorization of posts with tags for easier navigation.
